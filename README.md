# brickbreaker

This repository contains a simple HTML5 Brick Breaker game.

## Viewing the Game

- **Locally**: Open `index.html` in your browser.
- **GitHub Pages**: Rename or copy the game file to `index.html` (already done). Then in your repository settings enable GitHub Pages using the `main` branch (root or `/docs` folder).
  After enabling, visit `https://<your-username>.github.io/brickbreaker/` to play inline.

Feel free to customize the page or add assets as needed.
